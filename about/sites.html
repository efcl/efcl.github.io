運営しているブログやプログラミングで作成したものを公開するのに利用しているサイトまとめ

Web Scratch

メインとするブログです。 ソフトウェアの紹介や作成したソフトウェア、アドオン、JavaScriptなどについて書いています。

JSer.info

海外も含め最新のJavaScript情報について紹介しているサイトです。 なぜこのサイトを運営しているかはJSer.infoについてと世界のJavaScript情報を読もうなどに書いています。

Github

その他の多くのプログラムはGithubで公開しています。 このサイトについてもGithubにソースコードを公開しています。
